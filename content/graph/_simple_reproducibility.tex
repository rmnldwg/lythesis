    The four simple graphs we tested in this section for illustrative purposes can be reproduced using the \repolink{lynference} repository. In it, one tagged commit exists for the four graphs and the \href{https://github.com/rmnldwg/lynference#readme}{\faIcon{info-circle} \texttt{README.md}} file in this repository explains how one can download the data our pipeline runs produced and how to reproduce an experiment yourself.

    \begin{itemize}
        \item \textbf{Graph 1} (no connection btw. II and III): \href{https://github.com/rmnldwg/lynference/releases/tag/simple-nocon-v1}{\faIcon{tags} \texttt{simple-nocon-v1}}
        \item \textbf{Graph 2} (from II to III): \href{https://github.com/rmnldwg/lynference/releases/tag/simple-IItoIII-v1}{\faIcon{tags} \texttt{simple-IItoIII-v1}}
        \item \textbf{Graph 3} (from III to II): \href{https://github.com/rmnldwg/lynference/releases/tag/simple-IIItoII-v1}{\faIcon{tags} \texttt{simple-IIItoII-v1}}
        \item \textbf{Graph 4} (both directions): \href{https://github.com/rmnldwg/lynference/releases/tag/simple-both-v1}{\faIcon{tags} \texttt{simple-both-v1}}
    \end{itemize}
